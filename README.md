# Asteroids

## Description
My Asteroids-like game with proper thrust physics.

![](thumbnail.png)

## Requirements
[Processing](https://processing.org) is required to run the game.

## Quick start
1. Open Processing IDE.
2. Open `asteroids.pde`.
3. Run the game.

## Controls
A, D keys turn left or right.

W key to thrust.

X key to shoot lasers.
